# ESP32
En este repositorio estan los codigos de la programación de la ESP32 para comunicación.
Gateway de Comunicación WiFi/UDP (ESP32).
Actúa como puente (gateway) recibiendo datos por UART desde la STM32 y retransmitiéndolos vía WiFi usando el protocolo UDP a una estación de supervisión.
Tecnologías:
-Framework: ESP-IDF.
-RTOS: FreeRTOS para gestión de tareas.
